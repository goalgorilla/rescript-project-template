# ReScript Project Template

ReScript React starter template for Open Social.

Uses esbuild and serve to create a bundle

## Installation

```sh
yarn install
```

## Build

- Build: `yarn build`
- Clean: `yarn clean`
- Build & watch: `yarn start`

## Run

```sh
yarn dev
```
